Custom Filters

Como criar seus próprios filters?

Sintaxe de um custom filter

Formatação de data usando moment.js

angular.module('myApp', []).filter('myFormatDate', function() {
  return function(input) {
     if (typeof moment === 'function') { 
        if( typeof input !== 'undefined' && moment(input).isValid() ) {
           input = moment(input).format('MM/DD/YYYY [at] h:mm A');
        }
     }
     return input;
  };
})

Sintaxe de um custom filter

No html:

<article>
   <h1>{{ article.title }}</h1>
   <time>{{ article.date | myFormatDate }}</time>
</article>

Recebendo parâmetros

Formatação de data usando moment.js

angular.module('myApp', []).filter('myFormatDate', function( stringFormat ) {
  return function(input) {
     if (typeof moment === 'function') { 
        if( typeof input !== 'undefined' && moment(input).isValid() ) {
           input = moment(input).format( stringFormat );
        }
     }
     return input;
  };
})

<article>
   <h1>{{ article.title }}</h1>

</article>

Custom filters para listagens

Custom filters para listagens

myApp.filter('axFilter', function($filter) {
  return function( list, filterSelected, textSearch ) {
    var outputList = [];
    
    if ( filterSelected === 'reviewed' ){
        angular.forEach(list, function(value, key){
          var obj = value;
          if(!obj.reviewed){
             this.push(obj);
          }
        }, outputList);
    }
    
    return $filter('filter')(outputList, textSearch);
  };
});

Custom filters para listagens

Html:

<select ng-model="fltr.select" ng-options="sts.name for sts in status">
</select>

<input type="text" ng-model="fltr.search">

<ul ng-repeat="item in items | axFilter:fltr.select:fltr.search">
   <li>{{ item.name }}</li>
</ul>

Testes com Protractor

Como automatizar testes da sua aplicação AngularJS?

O que é Protractor?

É um framework de testes funcionais (End-to-end testing) escrito sobre o WebdriverJS (Selenium) que simula a navegação do usuário usando o browser.

Por que Protractor?

  • É mantido pela equipe do AngularJS
  • É fácil de aprender, tem sintaxe simplificada.
  • Usa o jasmine. Torna-se familiar para quem já fez unit testes.
  • Roda sobre o NodeJS. Você pode turbinar seu teste utilizando outros pacotes npm.

element() vs element.all()

Single element

element( by.binding('appName') );

Collection of elements

// clicks the 3th element
element.all( by.css('[ng-click="openPage()"]') ).get(2).click();

by.binding

In your test

element( by.binding('myModel') );

In your application

<span ng-bind="myModel"></span>
<!-- or -->
<span>{{myModel}}</span>

by.model

In your test

element( by.model('myModel') );

In your application

<input ng-model="myModel" />

by.repeater

In your test

element( by.repeater('user in users').row(0).column('name') );

In your application

<ul>
  <li ng-repeat="user in users">
      <span>{{user.name}}</span>
  </li>
</ul>

by.css

In your test

element( by.css('[ng-click="sendMail()"]') );

In your application

<button ng-click="sendMail()">Send mail!</button>

Searching elements best practices

  • Take advantage of AngularJS attributes using by.model, by.repeater, etc
  • Avoid using potential CSS attributes, mainly IDs and Classes.

Debbuging using elementexplorer

  1. Start your webdriver server:
    webdriver-manager start
  2. Run:
    /usr/local/lib/node_modules/protractor/bin/elementexplorer.js http://angularjs.org
  3. Press 'tab' and play with any element locator.
